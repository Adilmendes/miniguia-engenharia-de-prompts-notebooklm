- **Prompt inicial:** “Explique o conceito de Engenharia de Prompt para iniciantes.”

- **Dificuldade encontrada:** Respostas muito genéricas.  
- **Solução:** Refinar o prompt adicionando contexto (“considerando aplicações em educação e produtividade”).

## 📖 Miniguia de Estudo (Entrega Final)  

### 🔹 Resumo Estruturado  
Engenharia de Prompt é a prática de criar instruções claras e estratégicas para modelos de IA, visando obter respostas mais relevantes, criativas e úteis.  

### 🔹 Glossário  
- **Prompt:** Instrução ou comando dado à IA.  
- **Contexto:** Informações adicionais que ajudam a IA a entender melhor a tarefa.  
- **Chain-of-Thought:** Técnica de induzir raciocínio passo a passo.  
- **Zero-shot / Few-shot:** Estratégias de fornecer exemplos ou não fornecer nenhum.  
- **Delimitadores:** Símbolos ou estruturas que organizam melhor a entrada.  
- **Frameworks de Prompt:** Estruturas como C.E.R.T.O ou 5-layer framework que ajudam a padronizar prompts.  

### 🔹 Conjunto de Prompts Reutilizáveis  

**Prompts recorrentes (da imagem):**  
1. O que é o método C.E.R.T.O para estruturar prompts eficazes?  
2. Como as técnicas de Chain-of-Thought e Self-Consistency melhoram o raciocínio?  
3. Quais são as principais diferenças entre engenharia de prompt e engenharia de contexto?  

**Outros prompts estratégicos:**  
- Explique [conceito] em linguagem simples, como se fosse para um estudante do ensino médio.  
- Liste prós e contras de [tema] em formato de tabela.  
- Crie um resumo estruturado com tópicos e subtópicos sobre [assunto].  
- Sugira 5 perguntas de revisão sobre [tema].  
- Mostre exemplos de aplicação prática de [conceito] em projetos reais.
